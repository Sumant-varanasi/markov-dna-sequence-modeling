## Abstract

This project investigates **higher-order Markov chains** as generative models for genomic DNA sequences.

Using the *E. coli* K-12 reference genome, we train first- through fourth-order Markov models to capture local nucleotide dependencies and generate synthetic DNA preserving statistical properties of real genomes.

We evaluate generated sequences through:

- Base composition preservation  
- k-mer entropy analysis  
- KL divergence between real and synthetic distributions  
- Next-nucleotide prediction accuracy  
- Motif frequency preservation  
- Sliding-window GC content structure

Our experiments show **trinucleotide (order-3) models** provide the strongest tradeoff between predictive power, statistical fidelity, and computational efficiency.

---

## Key Result

| Order | Accuracy | KL Divergence | Entropy Match |
|------|----------|---------------|---------------|
| 1 | 0.3034 | 0.001696 | High |
| 2 | 0.3223 | 0.003552 | High |
| **3** | **0.3322** | **0.006171** | **Best Tradeoff** |
| 4 | 0.3380 | 0.012961 | Slight Overfit |

Order-3 Markov chains reproduce empirical genomic statistics while remaining computationally compact.

## Method

Given an order-\(n\) model,

\[
P(x_t | x_{t-n}, ..., x_{t-1})
\]

transition probabilities are estimated from observed n-gram counts:

\[
P(a \mid s)=
\frac{C(sa)}{\sum_{b \in \{A,C,G,T\}} C(sb)}
\]

Synthetic DNA is generated by iterative sampling from learned transition distributions.
